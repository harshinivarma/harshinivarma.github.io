Covid-19 Desktop

Desktop Application of Corona Virus Tracking Dashboard every 5 mins update stats Covid-19 Desktop simplifies your way of seeing Corona virus cases around the globe.

Download for Windows (64bit)

By downloading, you agree to the CC0 1.0 Universal Applications Terms.

World COVID-19 Stats

Here you can see World's Corona Total Cases, Total Deaths, Total Recovered it will update every 5 mins and also you have table country wise cases, recovered, new confirmed and etc.,

Total india COVID-19 stats

As we (Developers) From india We've Added one special tab for india corona cases, TotalConfirmed cases, Recovered, Deaths, Today Cases , Actives Cases. etc.,

Web Technologies

We build This Application Using Electron Framework we can build Applications with HTML, CSS, javascript. Electron also uses Chromium V8 engine and Node.js
